#  Wings of Westeros

Welcome to **Wings of Westeros**, a high-octane, dragon-riding arcade game set in the world of the Seven Kingdoms. Take control of legendary dragons, dodge obstacles, battle the Night King's army, and conquer the skies!

##  Play Now
Simply clone the repository and open `index.html` in your web browser. No installation required!

##  Features

###  Legendary Dragons
Choose your mount from the greatest dragons in history, each with unique visuals:
*   **Balerion the Black Dread**
*   **Drogon**
*   **Caraxes (The Blood Wyrm)**
*   **Viserion**
*   **Meleys (The Red Queen)**

###  Explore the Seven Kingdoms
Fly through iconic locations with dynamic backgrounds and themed obstacles:
*   **House Stark** (The North)
*   **House Lannister** (Casterly Rock)
*   **House Targaryen** (Dragonstone)
*   **House Baratheon**, **Greyjoy**, **Tyrell**, **Martell**, and **Arryn**.

###  Combat System
This isn't just about flying!
*   **Shoot Fireballs**: Unleash dragon fire (Dracarys!) to destroy enemies and clear your path.
*   **Stamina Management**: Fireballs cost stamina. Manage your energy wisely!
*   **Boss Battles**: Face off against the terrifying **Night Dragon** in epic boss fights. Dodge ice blasts and strike back to survive.

##  Controls

| Action | PC (Keyboard/Mouse) | Mobile (Touch) |
| :--- | :--- | :--- |
| **Fly / Flap** | `Spacebar` or `Left Click` | Tap Screen |
| **Shoot Fire** | `K` Key or `Shoot Button` | Tap `Shoot Button` |

##  How to Run

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/wings-of-westeros.git
    ```
2.  **Navigate to the folder**
    ```bash
    cd wings-of-westeros
    ```
3.  **Launch the Game**
    *   Double-click `index.html` to open it in your default web browser.
    *   *Tip: For the best experience, use a modern browser like Chrome, Firefox, or Edge.*

##  Technologies Used
*   **HTML5 Canvas**: For high-performance 2D rendering.
*   **Vanilla JavaScript**: Core game logic, physics, and state management (No heavy frameworks!).
*   **CSS3**: Styling the UI and responsive design.
*   **Web Audio API**: Procedurally generated sound effects for an immersive experience.
